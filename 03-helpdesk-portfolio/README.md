# Helpdesk Lab Portfolio

**Series:** Kevtech IT Support — Entry Level Helpdesk (New Techs)
**Format:** Self-hosted lab environment · 14 modules · ~250 hours hands-on practice
**Environment:** VirtualBox · Windows Server 2016 · Windows 10 Pro

---

## Overview

This folder documents a complete, self-directed Helpdesk lab series simulating a real-world IT support environment. Each module covers a specific Helpdesk competency, from initial server setup to advanced remote support, software deployment, and ticketing workflows. The lab was built and run locally using VirtualBox, with no external cloud infrastructure.

All configurations, commands, and observations are documented in the notes for each module.

---

## Lab environment

| Component | Details |
|-----------|---------|
| Hypervisor | Oracle VirtualBox |
| Server OS | Windows Server 2016 (Domain Controller) |
| Client OS | Windows 10 Pro (domain-joined workstation) |
| Directory service | Active Directory Domain Services (AD DS) |
| Admin tools | RSAT, Server Manager, Group Policy Management Console |
| Ticketing system | Spiceworks (Help Desk) |
| Deployment tool | PDQ Deploy · PDQ Inventory |
